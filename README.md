## 📌 Overview

**Eco Zone** is an AI-driven environmental awareness application designed during the **EmpowerMe Hackathon** (July 21, 2025). The project focuses on automating environmental change detection without manual human input.

| **Theme** | Building Futures for MENA Youth |

---

## 🎯 Problem Statement

Environmental monitoring typically requires:
- ❌ Manual data collection
- ❌ Human intervention for change detection
- ❌ Delayed response to environmental issues
- ❌ Limited public awareness

---

## 💡 Solution

An automated system that:
- ✅ Uses AI to detect environmental changes
- ✅ Provides real-time alerts without human input
- ✅ Increases public awareness through accessible interfaces
- ✅ Encourages community action
